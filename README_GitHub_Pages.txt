Earth Greens Dashboard - GitHub Pages deployment

Files included:
- index.html

How to publish:
1. Create a new GitHub repository
2. Upload index.html to the repository root
3. Go to Settings > Pages
4. Under Build and deployment:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save
6. Wait a minute for the public URL to appear

Typical public URL:
https://YOUR_GITHUB_USERNAME.github.io/REPOSITORY_NAME/

Notes:
- This is a single-file offline dashboard
- No backend or database is required
- You can replace index.html later with a newer version
